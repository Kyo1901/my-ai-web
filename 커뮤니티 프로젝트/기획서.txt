== 나만의 커뮤니티 기획서 ==

커뮤니티 주제
- 소설/잡지 정보 공유

디자인 스타일
- 사이트 이름: Novel_Community
- 색상 테마: 파스텔톤의 푸른색
- 분위기: 친근하고 재밌는 분위기

특별한 기능 아이디어
- 소설/잡지 평점
- 신간/인기 소설/잡지 표시

페이지별 특징
- 메인 페이지
    - 신간/인기 소설 및 잡지 
    - 인기 장르 5개별로 소설 5 ~ 10개
    - 왼쪽 사이드에 소설 장르 및 잡지 종류 카테고리
    - 소설 및 잡지 검색

- 로그인 페이지
    - 이메일 입력 필드
    - 패스워드 입력 필드
    - 로그인 버튼

- 회원 가입 페이지
    - 이메일 입력 필드
    - 이메일 중복 검사 버튼
    - 비밀번호 입력 필드
    - 비밀번호 확인 입력 필드
    - 닉네임(이름)
    - 회원가입 버튼

- 게시물 목록:
    - 게시물 정보
        - 소설/잡지 제목
        - 소설/잡지 이미지
        - 저자, 출판사
        - 가격
        - 평점
        - 댓글 수
    - 왼쪽 사이드 바에 장르별 카테고리

- 게시물 상세:
    - 소설/잡지 제목
        - 소설/잡지 이미지
        - 저자, 출판사
        - 가격
        - 평점
        - 간단한 줄거리
    - 댓글
        - 댓글 작성자
        - 댓글 내용
        - 댓글 작성 시간
        - 작성자가 부여한 평점

- 댓글 기능:
    - 대댓글
    - 댓글 좋아요
    - 베스트 댓글 상단 고정

참고 사이트
- 교보 문고
- 네이버 카페
